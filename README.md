# MVK Transports Mecheri

Official website for MVK Transports — a professional transport and logistics service based in Mecheri, Tamil Nadu, operating since 2019.

## Owners

- **K. Madesh**
- **M. Gokul Vishnu**

## Features

- Hero section with stats and call-to-action
- Services: Goods Transportation, Relocation, Construction Logistics
- Live GPS tracking simulation (Mecheri → Salem)
- Modern fleet showcase (Mahindra Bolero Pickup & Eicher Truck)
- Instant price calculator based on distance, weight, and vehicle type
- Quick booking form with WhatsApp integration
- Testimonials slider with auto-rotation
- Contact section with location, phone, email, and working hours
- Floating WhatsApp button for quick contact
- Fully responsive design for mobile and desktop

## Tech Stack

- HTML5
- CSS3 (custom properties, animations, responsive grid)
- Vanilla JavaScript
- AOS (Animate On Scroll)
- Font Awesome icons
- Google Fonts (Inter)

## Getting Started

No build tools required. Just open `index.html` in a browser:

```bash
# Clone the repo
git clone https://github.com/your-username/MVK_TRANSPORTS.git

# Open in browser
open index.html
```
